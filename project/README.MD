# LexiAI Prototype - The Fine Printers

## Overview
This project builds a simple AI system to analyze NDA documents.

Features implemented:
- Summarization
- Risk Detection
- Jargon Explanation
- WIP

---

## Files
- main.ipynb → main notebook
- NDA-1.pdf → sample document
- NDA-2.pdf → sample document

Make sure all files are in the same folder.

---

## How to Run
1. Open main.ipynb in Jupyter
2. Run all cells from top to bottom
3. Outputs will be printed in the notebook

---

## Notes
- Uses Hugging Face models (T5, BART)
- Includes a trained Logistic Regression model for risk detection
